# 1. 프로젝트 개요
- 프로젝트명: Bloger(블로거)
- 목적: 최소한의 입력(사진, 단순 키워드/메모)만으로 자연스러운 방문 소감, 후기, 블로그 리뷰 초안을 생성해주는 웹 서비스. 글쓰기 접근성을 높이고 편의성을 극대화한다.

# 2. 목표 사용자
- 블로그나 방문 후기를 남기고 싶으나 긴 글 작성을 부담스러워하는 사용자
- 방문한 장소(식당, 카페, 미용실 등)에 대한 기록과 소감을 쉽고 빠르게 남기고 싶은 사람

# 3. 핵심 기능
- 업체 카테고리(식당, 카페, 미용실 등) 선택
- 사진 파일 업로드 및 첨부 기능
- 리뷰 소스 입력(키워드, 짧은 문장, 메모 형태)
- AI 기반 후기/초안 자동 생성
- 생성된 후기 초안 내용 확인, 수정 조율 보조 및 복사/저장

# 4. 페이지 구성
- **메인 페이지**: 서비스 소개, 시작하기(업체 카테고리 선택)
- **후기 작성 페이지**: 사진 첨부, 핵심 메모 및 키워드 입력 폼
- **후기 결과/수정 페이지**: 생성된 후기 초안 확인 및 에디팅, 완성본 복사

# 5. 컴포넌트 구성
- **Layouts**: `Header`, `Sidebar`, `Footer`
- **Common UI**: `Button`, `Input`, `Textarea`, `Card`, `Modal`, `List`
- **Domains**: `CategorySelector`, `ImageUploader`, `SourceInput`, `ResultViewer`

# 6. 상태 관리 구조
- **Editor State**: 선택된 카테고리, 업로드된 사진, 입력된 키워드/메모 내용 등 (Redux)
- **Result State**: 생성된 리뷰 원본 및 사용자가 수정 중인 결과물 (Redux)
- **UI/Local State**: 컴포넌트 내부 동작 모달 등 가벼운 상태 분리 관장
